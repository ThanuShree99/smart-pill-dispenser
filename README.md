#  Smart Pill Dispenser

##  Overview
An automated system that dispenses medicines at scheduled times using RTC and a servo motor.

---

##  Features
- Time-based dispensing
- Buzzer alert
- Servo motor control
- LCD display

---

##  Components
- Arduino UNO
- RTC
- Servo Motor
- LCD 
- Buzzer
